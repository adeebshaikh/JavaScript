Description of various console functions

1.console.log():
The console.log() is a function that writes a message to log on the debugging console.

2.console.clear():
The console.clear() method clears the console if the environment allows it.

3.console.info():
The console.info() method outputs an informational message to the Web Console.

4.console.warn():
Outputs a warning message to the Web Console.

5.console.error():
Outputs an error message to the Web Console.

6.console.trace():
The console interface's trace() method outputs a stack trace to the Web Console.

7.console.time():
Starts a timer you can use to track how long an operation takes.

8.console.timeEnd():
Stops a timer that was previously started by calling console.time()
